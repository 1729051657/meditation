# 倒计时功能更新说明

## 更新内容

已成功将定时关闭功能改造为动态倒计时显示，实现了以下功能：

### 1. Vuex 状态管理
- 在 `store/modules/timer.js` 中添加了响应式的 `remainingSeconds` 状态
- 更新了 `startSleepTimer` 和 `restoreTimer` actions，每秒更新剩余时间
- 添加了 `UPDATE_REMAINING_SECONDS` mutation 来更新剩余秒数
- 提供了格式化的 `remainingTime` getter，支持显示小时、分钟和秒

### 2. SleepTimer 组件更新
- 集成了 Vuex 的 mapState、mapGetters 和 mapActions
- 使用 Vuex 管理的定时器状态替代组件内部状态
- 组件挂载时自动恢复定时器状态

### 3. 播放器页面更新
- 添加了计算属性来获取定时器状态
- 在定时器图标下方显示动态倒计时（格式：分:秒）
- 页面加载时自动恢复定时器状态
- 定时器相关操作都使用 Vuex actions

### 4. 迷你播放器组件更新
- 集成了 Vuex 的定时器状态
- 在播放器条上显示倒计时信息
- 组件挂载时自动恢复定时器状态

### 5. 首页集成
- 通过 MiniPlayer 组件自动显示倒计时
- 无需额外配置，定时器状态全局共享

## 功能特点

1. **实时更新**：倒计时每秒自动更新，无需手动刷新
2. **全局同步**：所有页面的倒计时显示保持同步
3. **状态持久化**：定时器状态保存在本地存储，应用重启后可恢复
4. **智能格式化**：
   - 小于1小时显示：分:秒（如 5:30）
   - 大于1小时显示：时:分:秒（如 1:30:45）
5. **最后提醒**：倒计时最后10秒会弹出提醒

## 测试方法

1. 进入播放器页面
2. 点击定时器图标（右下角）
3. 选择定时时长或自定义时间
4. 观察以下位置的倒计时显示：
   - 播放器页面的定时器图标下方
   - 迷你播放器的播放条上
   - 首页的迷你播放器（如果有音频在播放）

## 技术实现

- 使用 Vuex 集中管理定时器状态
- 使用 setInterval 每秒更新剩余时间
- 使用 localStorage 持久化定时器结束时间
- 使用 Vue 的响应式系统自动更新UI

## 注意事项

1. 定时器在后台也会继续运行
2. 定时结束时会自动停止音频播放
3. 取消定时会清除所有相关状态
4. 页面刷新或应用重启后会自动恢复定时器
