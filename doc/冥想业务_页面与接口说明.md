### 文档目标
汇总首期冥想 App 的页面、已有接口（基于 Controller）、数据结构与验收点，指导前后端联动与排期。

### 约定
- 所有列表接口遵循 RuoYi 分页规范：`pageNum`、`pageSize`、`orderByColumn`、`isAsc`。
- 统一返回结构 `R`/`TableDataInfo`（与框架一致）。
- 资源字段统一引用 `sys_oss.oss_id`（图片/音频）。
- 多租户与审计字段按表结构落库（已在 `script/sql/ry_meditation.sql` 定义）。

### 关键实体
- 分类 Category（`mg_category`）
- 系列 Series（`mg_series`）
- 单集 Track（`mg_track`，含 `series_id`、`category_id`、`audio`、`duration_sec`）
- 知识 Article（`mg_article`）
- 标签与关系（`mg_tag`、`mg_content_tag`）
- 推荐位与内容（`mg_recommend_slot`、`mg_recommend_item`）
- 横幅 Banner（`mg_banner`）
- 搜索与热词（`mg_search_history`、`mg_hot_keyword`）
- 收藏与播放（`mg_favorite`、`mg_play_history`、`mg_series_progress`）

## 页面与接口映射（尽量复用已有 Controller）

### 1. 首页 Home
UI 模块：
- 分类快捷入口（放松减压/改善睡眠/提升专注/情绪调节）
- 模块「冥想练习」（系列/单集卡片，含时长/系列节数）
- 模块「冥想推荐」（单集宫格）
- 模块「冥想知识」（文章列表）
- 横幅 Banner、右上角搜索入口、底部迷你播放器

接口（组合调用）：
- 分类入口
  - GET `/meditation/category/list?parentId=0&status=0` → 一级分类
- 推荐位与内容（按页面/频道获取）
  - GET `/meditation/recommendSlot/list?page=home&status=0`
  - GET `/meditation/recommendItem/list?slotId={slotId}&status=0`（返回 series/track/article 的混合卡片）
- 横幅
  - GET `/meditation/banner/list?page=home&status=0`
- 文章（知识）
  - GET `/meditation/article/list?status=0&pageNum=1&pageSize=10`
- 迷你播放器（前端本地状态 + 播放记录接口，见“播放与进度”）

备注：若需单次聚合，可新增轻量 `HomeController` 封装以上数据；当前阶段按上述多接口并发获取即可。

### 2. 分类页 Category Feed（以“放松减压”为例）
UI 模块：
- 顶部 chips/标签切换（大类/子类/标签）
- 「为你推荐」（系列卡）
- 「全部冥想」（单集宫格，展示“10分钟”等角标）

接口：
- 分类树/子类：GET `/meditation/category/list?parentId={id}&status=0`
- 推荐位（分类页频道）：GET `/meditation/recommendSlot/list?page=category&status=0` + `/meditation/recommendItem/list?slotId={slotId}&status=0`
- 单集列表（按分类）：GET `/meditation/track/list?categoryId={id}&status=0&pageNum=1&pageSize=20`
- 可选系列列表：GET `/meditation/series/list?categoryId={id}&status=0`

### 3. 系列详情 Series Detail（含小节列表）
- 系列详情：GET `/meditation/series/{id}`
- 小节列表：GET `/meditation/track/list?seriesId={id}&status=0&orderByColumn=order_index&isAsc=asc`
- 进度：GET `/meditation/seriesProgress/list?seriesId={id}&userId={userId}`（或直接查 `/seriesProgress/{id}` 视实现而定）

### 4. 知识列表/详情 Article
- 列表：GET `/meditation/article/list?status=0&pageNum=1&pageSize=20`
- 详情：GET `/meditation/article/{id}`

### 5. 搜索 Search
- 结果（按需多次请求或后端聚合）：
  - 单集：GET `/meditation/track/list?title={kw}&status=0`
  - 系列：GET `/meditation/series/list?title={kw}&status=0`
  - 文章：GET `/meditation/article/list?title={kw}&status=0`
- 历史：
  - 列表：GET `/meditation/searchHistory/list?userId={userId}`
  - 新增：POST `/meditation/searchHistory`
  - 删除：DELETE `/meditation/searchHistory/{ids}`
- 热词：GET `/meditation/hotKeyword/list?status=0`

### 6. 收藏 Favorites
- 列表：GET `/meditation/favorite/list?userId={userId}&targetType=track|series|article`
- 新增：POST `/meditation/favorite`（Body 含 `userId`、`targetType`、`targetId`）
- 取消：DELETE `/meditation/favorite/{ids}` 或按业务提供批量取消

### 7. 播放与进度 Playback
- 开始/新增记录：POST `/meditation/playHistory`（`userId`、`trackId`、`progressSec=0`）
- 进度上报：PUT `/meditation/playHistory`（更新 `progressSec`、`isCompleted`、`lastPlayTime`）
- 最近播放/继续播放：GET `/meditation/playHistory/list?userId={userId}&orderByColumn=last_play_time&isAsc=desc&pageSize=1`
- 系列进度：
  - 查询：GET `/meditation/seriesProgress/list?userId={userId}&seriesId={seriesId}`
  - 更新：PUT `/meditation/seriesProgress`（`completedCount`、`lastTrackId`）

## 管理端（B 端）常用 CRUD（均已用模板生成，可直接使用）
> 路径均为：`/meditation/{资源}`，支持 `list/export/{id}/POST/PUT/DELETE`。

- 分类：`/meditation/category`
- 系列：`/meditation/series`
- 单集：`/meditation/track`
- 文章：`/meditation/article`
- 标签：`/meditation/tag`
- 内容-标签：`/meditation/contentTag`
- 推荐位：`/meditation/recommendSlot`
- 推荐位内容：`/meditation/recommendItem`
- 横幅：`/meditation/banner`
- 搜索历史：`/meditation/searchHistory`
- 热搜关键词：`/meditation/hotKeyword`
- 收藏：`/meditation/favorite`
- 播放记录：`/meditation/playHistory`
- 系列进度：`/meditation/seriesProgress`

筛选字段（示例，具体以各 Bo 为准）：
- CategoryBo：`parentId`、`name`、`status`
- SeriesBo：`categoryId`、`title`、`status`
- TrackBo：`seriesId`、`categoryId`、`title`、`status`

- RecommendSlotBo：`code`、`page`、`status`
- RecommendItemBo：`slotId`、`contentType`、`status`
- BannerBo：`page`、`status`、`startTime/endTime`
- SearchHistoryBo：`userId`、`keyword`
- HotKeywordBo：`status`、`startTime/endTime`
- FavoriteBo：`userId`、`targetType`
- PlayHistoryBo：`userId`、`trackId`

## 前端实现要点（Plus-UI / H5）
- 分页参数按 RuoYi 规范；统一在请求层封装。
- 图片懒加载；音频断点续播，路由销毁或每 5s 上报进度。
- 卡片组件复用：系列/单集卡（含角标“10分钟”“系列·N节”）。
- 继续播放：进入首页时尝试查询最近未完成记录并展示迷你播放器。

---

## 验收清单（关键）
- 首页加载 ≤ 1.5s（并发拉取分类/推荐/横幅/文章）。
- 分类页筛选与分页正确，单集角标时长与 `duration_sec` 一致（向上取整到分钟）。
- 系列详情小节排序按 `order_index` 正确；进度完成数与播放记录一致。
- 搜索：历史记录增删改查、热词有效期、结果展示一致。
- 收藏：幂等，重复收藏不报错，列表状态正确。
- 播放：进度上报可靠，完成阈值默认 90%（可在前端控制）。
