### 小程序改造清单（冥想版）

目录调整
- pages：仅保留 `pages/login/login`、`pages/mg/home/index`、`pages/mg/category/index`、`pages/mg/search/index`、`pages/profile/index`
- subPages：新增 `subPages/mg/series/detail`、`subPages/mg/article/detail`、`subPages/mg/player/index`
- 删除与“巡检/任务/文件/隐患/活动”等不相关页面与路由（见 `pages.json` 已清理）

接口封装
- 新增 `api/` 下冥想模块：`category.js`、`series.js`、`track.js`、`article.js`、`recommend.js`、`banner.js`、`search.js`、`favorite.js`、`play.js`
- 全部基于现有 `utils/request.js`，遵循 RuoYi 分页参数

登录改造
- `api/auth.js`：`inspectionWechatLogin` 改为仅 `code` 登录（不需手机号）
- `pages/login/login.vue`：按钮触发 `wxLoginSimple()`，调用后端 `/xcx/auth/login` 完成登录，成功后进入冥想首页

全局
- `main.js` 注入 `Vue.prototype.$baseUrl` 用于拼接 OSS 下载地址

页面说明
- 首页：并发加载 分类/推荐位/横幅/文章；卡片点击跳转系列详情/播放/文章详情
- 分类页：chips 切换子类；单集宫格无限加载
- 搜索页：热词 + 单集/系列/文章 tabs
- 系列详情：小节按 `order_index` 排序，点击播放
- 播放器：定时上报进度，播放结束标记完成

后端联调注意
- `/xcx/auth/login` 支持 code 登录；返回 `{ access_token, user }`
- 资源字段（图片/音频）返回 OSS 对象 id；前端以 `${baseUrl}/system/oss/download/{ossId}` 访问（按网关调整）
- 所有 `/meditation/*` 列表接口支持 `status=0`、分页、排序
