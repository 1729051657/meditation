# 播放功能说明

## 功能概述

播放器支持完整的音频播放功能，包括自动连续播放、多种播放模式、播放历史记录等。

## 核心功能

### 1. 自动连续播放 ✅

当前音频播放完成后，会自动播放下一首：

- **实现位置**: `onEnded` 事件处理
- **执行流程**:
  1. 标记当前音频为已完成
  2. 根据播放模式决定下一首
  3. 自动开始播放

### 2. 播放模式

支持三种播放模式：

#### 2.1 列表循环（默认）
- 按顺序播放列表中的音频
- 播放到最后一首后，自动从第一首开始

#### 2.2 单曲循环
- 重复播放当前音频
- 播放结束后自动重新开始

#### 2.3 随机播放
- 随机选择下一首播放
- 不会连续播放同一首

### 3. 播放控制

#### 基本控制
- **播放/暂停**: 点击中间的播放按钮
- **上一曲**: 点击左侧的上一曲按钮
- **下一曲**: 点击右侧的下一曲按钮
- **进度调整**: 拖动进度条

#### 播放列表
- 点击播放列表按钮查看当前列表
- 点击列表中的任意歌曲直接播放

#### 播放模式切换
- 点击播放模式按钮切换模式
- 模式会保存到本地，下次打开时自动恢复

### 4. 播放历史

自动记录播放历史，包括：
- 播放的音频ID
- 播放进度
- 是否完成播放

### 5. 后台播放

支持后台播放功能：
- 退出播放器页面后继续播放
- 在首页显示迷你播放器
- 系统通知栏显示播放控制

## 使用说明

### 基本播放流程

1. **进入播放器**
   - 点击任意音频进入播放器
   - 自动加载音频信息和播放列表

2. **开始播放**
   - 点击播放按钮开始播放
   - 音频会自动连续播放

3. **切换播放模式**
   - 点击播放模式按钮
   - 在列表循环、单曲循环、随机播放之间切换

4. **查看播放列表**
   - 点击播放列表按钮
   - 查看当前播放列表
   - 点击任意歌曲切换播放

## 技术实现

### 播放完成处理

```javascript
audioContext.onEnded(() => {
  // 标记为已完成
  this.markAsCompleted()
  
  // 根据播放模式决定下一步
  if (this.playMode === 'single') {
    // 单曲循环
    this.audioContext.seek(0)
    this.audioContext.play()
  } else if (this.playMode === 'random') {
    // 随机播放
    this.playRandom()
  } else {
    // 列表循环
    this.next()
  }
})
```

### 播放模式切换

```javascript
togglePlayMode() {
  const modes = ['list', 'single', 'random']
  const currentIndex = modes.indexOf(this.playMode)
  const nextIndex = (currentIndex + 1) % modes.length
  this.playMode = modes[nextIndex]
  
  // 保存到本地
  uni.setStorageSync('playMode', this.playMode)
}
```

### 随机播放实现

```javascript
playRandom() {
  let randomIndex
  do {
    randomIndex = Math.floor(Math.random() * this.playlist.length)
  } while (randomIndex === this.currentIndex)
  
  this.playTrackAtIndex(randomIndex)
}
```

## 注意事项

1. **播放列表加载**
   - 根据 seriesId 或 categoryId 加载相关音频
   - 最多加载100首音频

2. **播放进度保存**
   - 每10秒自动保存播放进度
   - 暂停时立即保存进度

3. **播放模式持久化**
   - 播放模式保存在本地存储
   - 下次打开时自动恢复
