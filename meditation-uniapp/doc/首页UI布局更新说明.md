# 首页UI布局更新说明

## 更新概述

根据UI设计图，对首页进行了全面的布局和样式调整，使其更符合设计规范，提升视觉效果和用户体验。

## 主要更新内容

### 1. 顶部区域调整
**移除导航栏组件**：
- 不再使用 `tn-nav-bar` 组件
- 直接在页面顶部显示问候语和搜索图标

**问候语样式**：
- 字体大小：从 36rpx 增加到 48rpx
- 字体粗细：从 500 增加到 600
- 位置：左上角，与搜索图标形成左右布局

**搜索图标**：
- 位置：右上角
- 样式：半透明白色背景，圆形设计
- 阴影：添加轻微阴影效果

### 2. 背景图片优化
**透明度调整**：
- 从 0.6 调整为 1.0
- 背景图片完全显示，不再半透明

### 3. 功能按钮区域优化
**按钮样式**：
- 移除多余的 `feature-box` 包装
- 图标容器：从 120rpx × 120rpx 调整为 100rpx × 140rpx（竖着长方形）
- 圆角：从 24rpx 调整为 20rpx
- 阴影：增强阴影效果，从 0.08 调整为 0.1

**布局修复**：
- 图标和文字现在都在同一个 `feature-icon-wrapper` 容器内
- 容器使用 `flex-direction: column` 垂直排列
- 图标下方添加 6rpx 间距（从 8rpx 减小），然后显示文字

**尺寸优化**：
- 图标尺寸：从 60rpx × 60rpx 调整为 50rpx × 50rpx
- 文字大小：从 20rpx 调整为 18rpx
- 整体更紧凑，间距更小

**交互效果**：
- 添加点击动画：`transform: scale(0.95)`
- 过渡效果：`transition: all 0.3s ease`
- 文字样式：字体大小调整为 20rpx，粗细 500，居中对齐

**间距调整**：
- 上下内边距：从 20rpx/40rpx 调整为 40rpx/60rpx
- 图标与文字间距：从 16rpx 调整为 20rpx

### 4. 冥想练习区域优化
**卡片尺寸**：
- 宽度：从 360rpx 调整为 500rpx（符合设计图500px）
- 高度：从 220rpx 调整为 339rpx（符合设计图339px）
- 圆角：从 20rpx 调整为 32rpx

**布局重构**：
- 移除原有的 flex 布局，改用绝对定位
- 标题位置：左上角，距离边缘 32rpx
- 底部控制条：新增半透明底部条，高度 80rpx

**底部控制条设计**：
- 背景：半透明白色 `rgba(255, 255, 255, 0.15)`
- 毛玻璃效果：`backdrop-filter: blur(20rpx)`
- 内容布局：左侧显示时长，右侧显示播放按钮
- 时长样式：白色文字，28rpx 字体，500 粗细
- 播放按钮：60rpx × 60rpx，半透明白色背景

**底部控制条修复**：
- 位置：距离底部 24rpx，距离左侧 20rpx
- 尺寸：宽度 460rpx，高度 88rpx
- 背景：蓝色半透明 `rgba(121, 177, 218, 0.6)`
- 圆角：20rpx
- 毛玻璃效果：`backdrop-filter: blur(1px)`
- 不再贴紧底部边缘，而是悬浮在卡片内部

**内容样式**：
- 标题字体：从 36rpx 调整为 48rpx，粗细 600
- 添加文字阴影：`text-shadow: 0 2rpx 4rpx rgba(0, 0, 0, 0.1)`
- 背景渐变：调整为更柔和的蓝色渐变

**间距调整**：
- 卡片间距：从 24rpx 调整为 30rpx
- 区域下边距：从 40rpx 调整为 50rpx

## 设计原则

### 1. 视觉层次
- 使用不同的字体大小和粗细建立信息层次
- 通过阴影和圆角增强立体感
- 合理的间距确保内容呼吸感

### 2. 交互反馈
- 所有可点击元素都有视觉反馈
- 使用缩放动画提升交互体验
- 过渡效果使动画更流畅

### 3. 一致性
- 统一的圆角半径（20rpx-24rpx）
- 一致的阴影风格
- 协调的颜色搭配

## 技术实现

### 1. CSS优化
- 使用 `transition` 实现平滑动画
- 使用 `transform` 实现交互效果
- 使用 `backdrop-filter` 实现毛玻璃效果

### 2. 响应式设计
- 使用 rpx 单位确保不同设备适配
- 合理的间距和尺寸设计
- 触摸友好的交互区域
