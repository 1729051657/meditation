# 首页滚动功能说明

## 功能概述

首页的冥想练习和冥想推荐区域都支持左右横向滚动，用户可以滑动查看更多内容，提升浏览体验。

## 滚动区域

### 1. 冥想练习区域
**位置**：功能按钮下方
**滚动方向**：左右横向滚动
**内容类型**：冥想练习卡片
**卡片样式**：渐变背景，包含标题、时长和播放按钮

### 2. 冥想推荐区域
**位置**：冥想练习区域下方
**滚动方向**：左右横向滚动
**内容类型**：推荐内容卡片
**卡片样式**：白色背景，包含封面图、标题和时长

## 技术实现

### Scroll-view 组件配置
```vue
<scroll-view 
  class="meditation-cards" 
  scroll-x 
  :show-scrollbar="false" 
  enhanced
>
  <!-- 内容列表 -->
</scroll-view>
```

**属性说明**：
- `scroll-x`：启用横向滚动
- `:show-scrollbar="false"`：隐藏滚动条，保持界面美观
- `enhanced`：启用增强滚动，提升滚动性能

### 样式配置
```scss
.meditation-cards {
  padding-left: 30rpx;  // 左侧留白
  
  .meditation-card-list {
    display: flex;       // 横向排列
    gap: 20rpx;         // 卡片间距
    padding-right: 30rpx; // 右侧留白，确保最后一个卡片完全可见
  }
}
```

## 滚动体验优化

### 1. 卡片尺寸固定
- **冥想练习卡片**：340rpx × 200rpx
- **推荐内容卡片**：280rpx × 自适应高度
- 使用 `flex-shrink: 0` 防止卡片被压缩

### 2. 间距设计
- 卡片间距：20rpx
- 左右边距：30rpx
- 确保滚动时有足够的视觉空间

### 3. 内容溢出处理
```scss
.card-title {
  display: -webkit-box;
  -webkit-line-clamp: 2;        // 最多显示2行
  -webkit-box-orient: vertical;
  overflow: hidden;
  text-overflow: ellipsis;      // 超出部分显示省略号
}
```

## 交互功能

### 1. 点击事件
- **冥想练习卡片**：`@click="goToMeditation(item)"`
- **推荐内容卡片**：`@click="goToRecommend(item)"`

### 2. 更多按钮
- **冥想练习**：`@click="goToMore('meditation')"`
- **冥想推荐**：`@click="goToMore('recommend')"`

## 响应式设计

### 1. 不同屏幕适配
- 使用 rpx 单位，自动适配不同设备
- 卡片宽度固定，确保在不同设备上显示一致

### 2. 触摸优化
- 支持触摸滑动
- 滚动惯性，提升用户体验

## 性能优化

### 1. 虚拟滚动
- 只渲染可见区域的卡片
- 减少 DOM 节点数量

### 2. 图片懒加载
- 使用 `mode="aspectFill"` 优化图片显示
- 避免一次性加载所有图片

### 3. 滚动优化
- 启用 `enhanced` 属性
- 使用 CSS3 硬件加速

## 使用说明

### 1. 滚动操作
- **触摸滑动**：在卡片区域左右滑动
- **惯性滚动**：快速滑动后会有惯性效果
- **边界回弹**：到达边界时会有回弹效果

### 2. 内容浏览
- 滑动查看所有可用内容
- 点击卡片进入详情页面
- 点击"更多"查看完整列表
