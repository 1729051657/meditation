# 图标使用说明

## 导航栏图标

### 图标文件位置
所有导航栏图标都存放在 `/static/tabbar/` 目录下

### 图标命名规范
- `home.png` - 首页图标（未激活状态）
- `home-active.png` - 首页图标（激活状态）
- `profile.png` - 我的页面图标（未激活状态）
- `profile-active.png` - 我的页面图标（激活状态）

### 文件重命名说明
原来的中文文件名已经重命名为英文：
- `首页备份@2x.png` → `home.png`
- `首页active@2x.png` → `home-active.png`
- `我的@2x.png` → `profile.png`
- `我的active@2x.png` → `profile-active.png`

### 图标规格要求
- **尺寸**: 81x81px（推荐）
- **格式**: PNG（支持透明背景）
- **风格**: 冥想主题，简洁现代
- **颜色**: 
  - 未激活状态：灰色系 (#6B7280)
  - 激活状态：紫色系 (#7C3AED)

## 图标设计建议

### 首页图标
- 使用房子、冥想垫、莲花等冥想相关元素
- 线条简洁，易于识别
- 保持与整体设计风格一致

### 我的页面图标
- 使用用户头像、个人中心等元素
- 与首页图标风格保持一致
- 确保在小尺寸下清晰可见

## 配置说明

### pages.json 配置
```json
{
  "tabBar": {
    "list": [
      { 
        "pagePath": "pages/home/index", 
        "iconPath": "/static/tabbar/home.png", 
        "selectedIconPath": "/static/tabbar/home-active.png", 
        "text": "首页" 
      },
      { 
        "pagePath": "pages/me/index", 
        "iconPath": "/static/tabbar/profile.png", 
        "selectedIconPath": "/static/tabbar/profile-active.png", 
        "text": "我的" 
      }
    ]
  }
}
```

## 注意事项

1. **图标尺寸**: 建议使用 81x81px，确保在不同设备上显示清晰
2. **文件大小**: 单个图标文件建议不超过 10KB
3. **透明背景**: 使用 PNG 格式支持透明背景，更好地融入界面
4. **一致性**: 所有图标应保持统一的设计风格和视觉语言
5. **可访问性**: 确保图标在小尺寸下仍然清晰可识别
6. **文件命名**: 使用英文命名，避免中文字符可能导致的兼容性问题

## 图标资源

如果需要获取冥想主题的图标资源，可以考虑以下来源：
- 图标库：Feather Icons、Heroicons、Material Icons
- 设计资源：Figma Community、Dribbble
- 商业图标：Iconfinder、Flaticon

## 更新流程

1. 准备新的图标文件
2. 按照命名规范重命名文件
3. 替换 `/static/tabbar/` 目录下的对应文件
4. 测试在不同设备上的显示效果
5. 更新相关文档
